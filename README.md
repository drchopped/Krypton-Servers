# Krypton Servers

A sleek desktop app for managing Minecraft servers. Browse and install mods & modpacks from Modrinth and CurseForge, manage worlds, players, backups, and server properties — all locally, no cloud or account required.

## Features
- Create & manage multiple Minecraft servers (Vanilla, Paper, Purpur, Fabric, Forge, NeoForge)
- Browse & install mods and modpacks directly from Modrinth and CurseForge
- Built-in console, file manager, player management, and world management
- Server icon customization (upload, auto-grab from modpacks, reset)
- Backup & restore servers
- Java version detection & installation
- Works fully offline — no account or cloud dependency
